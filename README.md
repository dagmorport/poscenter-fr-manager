# POScenter FR Manager

GUI-утилита для подключения к фискальным регистраторам (ФР) **Poscenter Ритейл 01-ФМ** через SSH-туннель.

## Проблема

Кассы Poscenter работают под Linux и подключены к внутренней сети. Фискальный регистратор находится за SSH-туннелем, и для работы с ним через Windows-драйвер ДККТ необходимо пробросить порт. Ручное подключение через plink требует знания командной строки и занимает время.

## Решение

Приложение автоматизирует весь процесс:

1. Подключение к кассе по SSH
2. Отключение графического интерфейса (Xorg) на кассе — необходимо для работы ФР
3. Проброс порта FR (7778) на локальный порт (17778) через SSH-туннель
4. Готовый адрес `127.0.0.1:17778` для настройки драйвера ДККТ

## Возможности

- **GUI интерфейс** — выбор кассы из списка, ввод пароля, одна кнопка "Connect"
- **CLI режим** — подключение через `connect.ps1` с параметрами
- **Автоматическое отключение Xorg** — не нужно заходить на кассу вручную
- **Копирование адреса** — кнопка "Copy FR Address" копирует `127.0.0.1:17778` в буфер
- **Автообновление** — проверка обновлений с GitHub при запуске
- **Логирование** — все действия записываются в `logs/`

## Быстрый старт

### Зависимости

- Windows 10/11
- PowerShell 5.1+
- [plink.exe](https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html) (в комплекте)

### Настройка

Отредактируйте `config.json` — укажите IP-адреса ваших касс:

```json
{
  "fr_ip": "192.168.X.X",
  "fr_port": 7778,
  "local_port": 17778,
  "ssh_port": 22,
  "ssh_user": "root",
  "plink_path": "",
  "kassas": [
    {"name": "KASSA_1", "ip": "192.168.1.100"},
    {"name": "KASSA_2", "ip": "192.168.1.101"}
  ]
}
```

### Запуск

```
run.bat
```

1. Выберите кассу в списке
2. Введите пароль root
3. Нажмите **Connect**
4. Скопируйте адрес ФР кнопкой **Copy FR Address**
5. Вставьте в настройки драйвера ДККТ: `127.0.0.1:17778`

## CLI

```powershell
# Подключение к конкретной кассе
.\connect.ps1 -IP 192.168.1.100

# Подключение с выбором из списка
.\connect.ps1

# Отключение всех туннелей
.\disconnect.ps1

# Настройка SSH-ключа (без пароля)
.\setup-key.ps1
```

## Как это работает

```
[Windows]                    [Касса Poscenter]              [ФР]
    |                              |                          |
    |--- SSH (plink) ------------->|                          |
    |    -L 17778:FR:7778          |                          |
    |                              |--- FR protocol --------->|
    |<-- localhost:17778 ---------|                          |
    |                              |                          |
```

Драйвер ДККТ подключается к `127.0.0.1:17778`, туннель перенаправляет трафик на ФР через кассу.
